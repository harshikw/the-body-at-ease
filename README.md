# The Body at Ease (Next.js + Tailwind)

## Quick start
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Deploy
- Vercel: `npm i -g vercel && vercel` or push to GitHub and import in Vercel UI.

## Where to put images
- `public/logo.png` — your real logo (currently your uploaded logo if detected)
- `public/studio-hero.jpg` — hero photo (placeholder included)
- `public/og.jpg` — social share image (placeholder included)

## Edit content
- Main component: `app/TheBodyAtEase.jsx`
- SEO: `app/page.jsx` (metadata)
- Styles: `app/globals.css` (Tailwind included)
